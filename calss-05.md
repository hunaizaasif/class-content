## 📸 Image Generation Crash Course
We looked into the fundamentals that shape every great photograph:

**Lenses:**
How focal lengths (24mm, 50mm, 85mm, etc.) change perspective, distortion, and subject framing.

**Aperture (ƒ-stop):**
Controls light + depth of field. Wide aperture (ƒ1.8) = blurry background; narrow aperture (ƒ11) = everything sharp.

**Lighting Setups:**
Studio lights, softboxes, rim lights, and how placement affects mood and detail.

**Photography Styles:**
Portrait, landscape, product, cinematic, editorial — and how prompt wording guides the model toward each style.

## 🤖 AI-Driven vs AI-Native Development

### **AI-Driven Development**

Using AI to *supercharge* coding workflows.
You remain the builder; AI accelerates you.

**What it looks like:**
* Agents helping write, debug, refactor
* Faster delivery, fewer repetitive tasks

**Tools:**
* **Claude Code** (agentic coding)
* **Gemini CLI** (open-source AI workflows)
* **Zed** (AI-native code editor)

Result: ⚡ *10x faster development.*

### **AI-Native Development**
Building products where AI is the *core* of the architecture.

**Examples:**
* Autonomous agents performing tasks
* Self-learning recommendation systems
* AI-powered support systems

Future systems are designed *AI-first*, not AI-added-later.

## 🛠️ Gemini CLI Setup
1. Check Node.js version:
```bash
node -v
```
If below 20, download from nodejs.org.

2. Install Gemini CLI:
```bash
npm install -g @google/gemini-cli
```
3. Verify installation:
```bash
gemini -v
```
4. Authentication & First Launch:
```bash
gemini
```
