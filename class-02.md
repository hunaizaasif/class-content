## 🧠 **Fundamental Prompting Techniques**
We explored the **core methods to shape LLM responses**:
* **Zero-shot prompting** – just ask directly, no examples.
* **One-shot prompting** – give a single example to set the pattern.
* **Few-shot prompting** – use multiple examples to teach format and tone.
* **System prompting** – define how the AI should behave globally.
* **Role prompting** – make the model act as a specific expert.
* **Contextual prompting** – provide background info to improve accuracy.
~~------------------------------~~

## ⚙️ **Advanced Prompting Strategies**
We moved into advanced reasoning and control techniques for complex AI tasks:
* **🧩 Chain of Thought (CoT)** – guide AI to think step by step.
* **🔁 Self-Consistency** – use multiple reasoning paths and choose the most reliable result.
* **🔍 Step-Back Prompting** – ask general → then specific questions for grounded answers.
* **🤖 ReAct (Reasoning + Acting)** – combine logical thinking with tool usage or web search.
* **🌳 Tree of Thoughts (ToT)** – explore multiple reasoning branches, compare options, and merge the best ideas.
* **🧬 Mixture of Experts (MoE)** – your prompt can steer which internal “expert” module handles the task, so domain cues / clarity matter.
* **🧪 A/B Testing** – treat prompt variants like experiments — compare outputs, pick the stronger prompt based on metrics or human judgment.

These strategies help LLMs **reason better, reduce hallucinations**, and **produce structured, logical outputs**.
